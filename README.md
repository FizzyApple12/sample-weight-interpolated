# Sample Weight Interpolated

An add-on for Blender to add a variant of the Sample Weight tool that interpolates between vertices

## Usage

When in the Weight Paint interaction mode, do one of the following:
* Click and hold on the Sample Weight tool to bring out the options and select ``Sample Weight Interpolated``
* Click ``Weights > Sample Weight Interpolated``
* Use F3 to find ``Sample Weight Interpolated``
* Shift + Alt + X

you can then click to get the interpolated weight.

## Installation

1. Download the python file from the releases
2. Click ``Edit > Preferences...`` in Blender
3. Go to the ``Add-ons`` tab
4. Click the dropdown in the top-right corner and click ``Install from Disk...``
5. Select the python file downloaded from the releases
6. Click the ``Install from Disk`` button
